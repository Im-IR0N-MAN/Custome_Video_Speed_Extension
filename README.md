# Global Video Speed Controller

A lightweight Chromium extension to take full control of video playback speeds across the web. Adjust any HTML5 video from **0.5x to 4.0x** using a synchronized slider OR input field.

## Features
* **Global Control:** Works on YouTube, Netflix, Vimeo, and any site using HTML5 video.
* **Dual Interface:** Toggle speed via a slider or numeric input.
* **Persistent Speed:** Remembers the preferred playback rate using `chrome.storage`.
* **Zero Bloat:** Minimalist UI with no background tracking or external dependencies.

Since this extension is not currently on the Web Store, follow these steps to install it:

1. **Clone/Download** this repository to your local machine.
2. Open your browser and navigate to `chrome://extensions/`.
3. Enable **"Developer mode"** in the top-right corner.
4. Click **"Load unpacked"** and select the project folder.
5. (Optional) Click the "Puzzle" icon in your toolbar and **Pin** the extension for quick access.

## Usage
1. Click the extension icon in the top-right corner of your browser.
2. Adjust the speed using the slider or type a value between `0.5` and `4.0`.
3. The video on the current page will update instantly.
